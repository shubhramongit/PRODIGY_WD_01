# Responsive Landing Page

A fully responsive landing page with a fixed navigation bar that dynamically changes style on scroll and hover interactions.

## 🚀 Features

- Fixed navigation menu
- Navbar background change on scroll
- Hover effects on menu items
- Responsive layout (mobile-friendly)
- Clean UI design

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox / Media Queries)
- JavaScript (DOM Manipulation & Scroll Events)

## 📂 Project Structure

├── index.html  
├── style.css  
└── script.js  

## 📌 How It Works

- JavaScript listens for the window scroll event.
- A CSS class is toggled when the page is scrolled.
- Hover effects are handled using CSS transitions.

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in your browser

## 🎯 Learning Outcome

- Understanding scroll events
- DOM manipulation
- Responsive design principles
- Fixed navigation implementation
